# Zone Alert — curated zone data

Weekly snapshots of municipal speed-limit and school/playground-zone data,
normalized for the Zone Alert app. Each `data/<city>.json.gz` is a gzipped
JSON file (`schemaVersion`, `city`, `generated`, `zones[]`); `manifest.json`
lists all cities with sizes and checksums.

Source: each city's open data portal (Open Government Licence variants).
Data is reproduced with attribution to the respective municipalities:
Calgary, Edmonton, Lethbridge, Airdrie, St. Albert, Surrey, Victoria,
Coquitlam, Abbotsford, Saskatoon, Regina, Winnipeg, Mississauga, London,
Windsor, Markham, Québec, Gatineau, Sherbrooke, Trois-Rivières, Moncton,
Saint John, Fredericton.

Generated by the Zone Alert data pipeline; do not edit by hand.
